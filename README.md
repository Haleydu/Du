# Du漫画浏览器

> 通过安装插件的模式，观看漫画

# 功能简介
- 左到右阅读模式
- 右到左阅读模式
- 上到下阅读模式
- 卷轴阅读模式
- 收藏漫画、检查漫画更新
- 本地备份恢复、webdav云备份和恢复功能
-
